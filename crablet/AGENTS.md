# Workspace Usage Guide

## Agent Capabilities
- **Manager**: Coordinates tasks and breaks down complex requests.
- **Coder**: Writes, debugs, and tests code (Rust, Python, TypeScript).
- **Researcher**: searches knowledge base and external sources.
- **Reviewer**: Reviews code and content for quality and security.

## Workflow
1. User submits a request.
2. Router analyzes complexity and intent.
3. System 1 handles simple queries instantly.
4. System 2/3 handles complex tasks, potentially spawning a swarm.
5. Agents collaborate using the shared workspace.

## Best Practices
- Use clear, specific instructions.
- Provide context when switching topics.
- Use "check status" to see active swarm tasks.
