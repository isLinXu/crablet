# :globe_with_meridians: Web UI

Crablet ships with a modern React-based web interface for browser-based interaction.

## Starting the Web UI

```bash
# Start unified Web UI + API gateway
crablet serve-web --port 18790

# Open http://localhost:18790
```

## Features

- :white_check_mark: Real-time chat with WebSocket
- :white_check_mark: Cognitive thought visualization
- :white_check_mark: Dashboard with system metrics
- :white_check_mark: Knowledge base management
- :white_check_mark: Skill browser and installer
- :white_check_mark: Settings panel
- :white_check_mark: Activity timeline
- :white_check_mark: Dark/light theme

## Interface Sections

| Section | Description |
|:--------|:------------|
| **Chat** | Main conversation interface with thought visualization |
| **Dashboard** | System metrics, activity timeline, agent status |
| **Knowledge** | Document management, RAG queries, entity explorer |
| **Skills** | Installed skills browser, marketplace |
| **Settings** | Model selection, safety config, channel management |
| **Observability** | Traces, metrics, logs |

## Thought Visualization

The Web UI renders Crablet's cognitive process in real-time:

- **System 1** indicators for fast responses
- **System 2** reasoning chain visualization
- **System 3** meta-cognitive reflection panel
- Cognitive load meters per system
