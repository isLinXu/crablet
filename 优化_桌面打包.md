## 2. 生成的图标转换脚本

### 脚本清单

| 脚本 | 路径 | 平台 | 依赖 |
|---|---|---|---|
| `generate-icons.sh` | `scripts/generate-icons.sh` | macOS / Linux | `sips` (macOS) 或 `ImageMagick` |
| `generate-icons.ps1` | `scripts/generate-icons.ps1` | Windows | .NET System.Drawing（内置） |
| `generate-icons.py` | `scripts/generate-icons.py` | 跨平台 | Python 3 + Pillow |

### 生成目标格式

- **macOS**: `icon.iconset/` (16~1024) + `icon.icns`
- **Windows**: `icon.ico` (16, 32, 48, 64, 128, 256 全尺寸)
- **Tauri bundle**: `32x32.png`, `64x64.png`, `128x128.png`, `128x128@2x.png`, `icon.png`(1024×1024)
- **Windows Store**: `Square30x30Logo.png` ~ `Square310x310Logo.png`, `StoreLogo.png`
- **Android**: `mipmap-mdpi` ~ `mipmap-xxxhdpi` (48~192)
- **iOS**: `AppIcon.appiconset` + `Contents.json` (20~1024)

### 使用方式

```bash
# Shell (macOS/Linux)
bash scripts/generate-icons.sh [源PNG] [输出目录]

# PowerShell (Windows)
.\scripts\generate-icons.ps1 -SourcePng "C:\path\logo.png" -OutDir "desktop\icons"

# Python (跨平台，推荐)
python3 scripts/generate-icons.py [源PNG] [输出目录]
```

## 3. 生成分发打包脚本

### 脚本清单

| 脚本 | 路径 | 平台 | 功能 |
|---|---|---|---|
| `build-release.sh` | `scripts/build-release.sh` | macOS / Linux | 编译多平台 sidecar + Tauri 打包 |
| `build-release.ps1` | `scripts/build-release.ps1` | Windows | 编译多平台 sidecar + Tauri 打包 |

### 脚本流程

1. **读取版本**: 从 `crablet/Cargo.toml` 获取当前版本
2. **编译 sidecar**: 遍历 `aarch64-apple-darwin`, `x86_64-apple-darwin`, `x86_64-unknown-linux-gnu`, `aarch64-unknown-linux-gnu`, `x86_64-pc-windows-msvc`, `aarch64-pc-windows-msvc`
3. **复制到 binaries/**: 按 `crablet-<target>` 命名
4. **创建通用副本**: 当前平台无后缀的 sidecar（Tauri 运行时查找用）
5. **构建 Tauri**: `npm run tauri build` 或 `cargo tauri build`
6. **输出分发物**: 列出 DMG / EXE / AppImage / DEB / MSI 等文件

### 环境变量 / 参数控制

```bash
# 仅编译 sidecar，跳过 Tauri
BUILD_SIDEcar=true BUILD_TAURI=false bash scripts/build-release.sh

# 跳过特定平台 sidecar
SKIP_SIDEcar="x86_64-pc-windows-msvc aarch64-pc-windows-msvc" bash scripts/build-release.sh
```

```powershell
# PowerShell 版本
.\scripts\build-release.ps1 -BuildSidecar -SkipSidecar "x86_64-pc-windows-msvc"
```
