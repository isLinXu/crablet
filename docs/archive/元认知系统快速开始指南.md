# 元认知系统快速开始指南

## 概述

Crablet 的元认知系统为 Agent 提供了自我监控、反思、学习和优化的能力，使其能够从经验中学习并持续改进。

## 核心概念

### 元认知循环

```
┌─────────────┐
│   Monitor   │ ← 监控执行过程，收集指标
└──────┬──────┘
        ↓
┌─────────────┐
│  Reflector  │ ← 分析问题，诊断原因
└──────┬──────┘
        ↓
┌─────────────┐
│   Learner   │ ← 提取模式，学习知识
└──────┬──────┘
        ↓
┌─────────────┐
│  Optimizer  │ ← 应用改进，优化策略
└─────────────┘
```

### 关键组件

1. **Monitor（监控器）**
   - 实时监控任务执行
   - 收集性能指标（置信度、质量、资源使用）
   - 记录用户反馈

2. **Reflector（反思器）**
   - 诊断问题类型
   - 分析根本原因（使用 LLM）
   - 生成改进建议

3. **Learner（学习器）**
   - 提取任务模式
   - 学习错误模式
   - 记录成功策略
   - 管理知识库

4. **Optimizer（优化器）**
   - 应用改进建议
   - 优化策略选择
   - 追踪策略性能
   - 选择最佳策略

## 快速开始

### 1. 创建元认知控制器

```rust
use crablet::cognitive::{
    MetaCognitiveController, create_llm_client,
};
use crablet::config::Config;

#[tokio::main]
async fn main() -> anyhow::Result<()> {
    // 创建配置
    let config = Config::default();
    let llm = create_llm_client(&config).await?;
    
    // 创建元认知控制器
    let controller = MetaCognitiveController::new(llm).await?;
    
    // 使用控制器...
    
    Ok(())
}
```

### 2. 执行任务（带元认知监控）

```rust
use std::time::Instant;

let request = ExecutionRequest {
    task_id: "task-1".into(),
    task: "Write a hello world function".into(),
    context: vec![],
    start_time: Instant::now(),
};

let result = controller.execute_with_meta(request, |req| {
    // 这里执行实际的任务逻辑
    ExecutionResult {
        task_id: req.task_id.clone(),
        success: true,
        output: "```rust\nfn hello() { println!(\"Hello\"); }\n```".into(),
        confidence: 0.95,
        duration: Duration::from_millis(100),
        metrics: ExecutionMetrics::default(),
    }
}).await;

println!("Success: {}", result.success);
println!("Confidence: {:.2}", result.confidence);
```

### 3. 获取统计信息

```rust
let stats = controller.get_statistics().await;

println!("Total tasks: {}", stats.total_tasks);
println!("Successful: {}", stats.successful_tasks);
println!("Failed: {}", stats.failed_tasks);
println!("Avg confidence: {:.2}", stats.avg_confidence);
println!("Patterns extracted: {}", stats.patterns_extracted);
println!("Improvements applied: {}", stats.improvements_applied);
```

### 4. 导出学习到的知识

```rust
let knowledge = controller.export_knowledge().await.unwrap();

println!("Learned {} knowledge items:", knowledge.len());
for item in knowledge {
    println!("- {}: {} (confidence: {:.2})", 
        item.knowledge_type, item.content, item.confidence);
}
```

### 5. 集成用户反馈

```rust
// 记录用户反馈（0.0 - 1.0）
controller.integrate_feedback("task-1", 0.9).await.unwrap();

// 反馈会被用来改进未来的执行
```

## 高级用法

### 自定义配置

```rust
use crablet::cognitive::MetaConfig;
use std::time::Duration;

let custom_config = MetaConfig {
    monitor_interval: Duration::from_millis(50),
    max_feedback_history: 1000,
    max_patterns: 2000,
    learning_threshold: 0.7,  // 只学习置信度 > 0.7 的任务
    enable_auto_optimization: true,
    optimization_interval: Duration::from_secs(60),
};

let controller = MetaCognitiveController::with_config(llm, custom_config).await?;
```

### 启动后台自动优化

```rust
// 启动后台优化循环
controller.start_background_optimization().await.unwrap();

// Agent 会在后台定期：
// 1. 分析学习到的模式
// 2. 应用改进建议
// 3. 优化策略选择
// 4. 更新知识库
```

### 创建自定义执行器

```rust
async fn custom_task_executor(
    request: &ExecutionRequest,
    // 其他依赖...
) -> ExecutionResult {
    // 1. 执行实际任务
    let output = execute_task(&request.task).await?;
    
    // 2. 计算置信度
    let confidence = calculate_confidence(&output);
    
    // 3. 收集指标
    let metrics = ExecutionMetrics {
        confidence,
        quality_score: calculate_quality(&output),
        resources: ResourceMetrics {
            duration_ms: 100,
            memory_bytes: 1024 * 1024,
            cpu_ms: 50,
            tokens_used: 100,
        },
        success: confidence > 0.5,
        error: None,
    };
    
    ExecutionResult {
        task_id: request.task_id.clone(),
        success: metrics.success,
        output,
        confidence,
        duration: Duration::from_millis(100),
        metrics,
    }
}

// 使用自定义执行器
let result = controller.execute_with_meta(request, custom_task_executor).await;
```

```rust
let knowledge = controller.export_knowledge().await.unwrap();

    .into_iter()
    .filter(|k| k.confidence > 0.7)
```

### 3. 监控和调优

```rust
// 定期检查统计信息
tokio::spawn(async move {
    let mut interval = tokio::time::interval(Duration::from_secs(300));
    loop {
        interval.tick().await;
        let stats = controller.get_statistics().await;
        
        if stats.failed_tasks > stats.successful_tasks / 10 {
            // 失败率过高，需要调整
            warn!("High failure rate detected: {:.1}%", 
                stats.failed_tasks as f64 / stats.total_tasks as f64 * 100.0);
        }
    }
});
```

### 4. 结合 Skills 系统


## 故障排查

### 问题 1: 学习没有效果

**可能原因：**
- 学习阈值设置过高
- 任务成功率太低
- 模式数量达到上限

**解决方案：**
```rust
// 降低学习阈值
let config = MetaConfig {
    learning_threshold: 0.5,  // 从 0.8 降低到 0.5
    max_patterns: 5000,        // 增加模式数量
    ...
};
```

### 问题 2: 性能下降

**可能原因：**
- 知识库过大
- 频繁的反思和优化
- LLM 调用过多

**解决方案：**
```rust
let config = MetaConfig {
    max_feedback_history: 500,      // 减少历史记录
    max_patterns: 500,              // 减少模式数量
    enable_auto_optimization: false, // 禁用自动优化
    ...
};

// 手动控制优化频率
tokio::spawn(async move {
    let mut interval = tokio::time::interval(Duration::from_secs(3600));
    loop {
        interval.tick().await;
        // 手动触发优化
    }
});
```

### 问题 3: 内存占用过高

**可能原因：**
- 知识库积累过多
- 反馈历史过长

**解决方案：**
```rust
// 定期清理
let controller = controller.clone();
tokio::spawn(async move {
    let mut interval = tokio::time::interval(Duration::from_secs(86400));
    loop {
        interval.tick().await;
        // 导出重要知识
        let knowledge = controller.export_knowledge().await.unwrap();
        save_to_disk(&knowledge).await;
        
        // 重置控制器（清理旧数据）
        let config = Config::default();
        let llm = create_llm_client(&config).await.unwrap();
        controller = MetaCognitiveController::new(llm).await.unwrap();
    }
});
```

## 性能优化建议

### 1. 使用缓存

```rust
use std::collections::HashMap;
use tokio::sync::RwLock;

let cache: Arc<RwLock<HashMap<String, Pattern>>> = Arc::new(RwLock::new(HashMap::new()));

// 在 Learner 中使用缓存
let cache = cache.clone();
tokio::spawn(async move {
    let mut cache = cache.write().await;
    cache.insert(pattern_id.clone(), pattern);
});
```

### 2. 批量处理

```rust
// 批量处理多个任务
let mut handles = vec![];
for task in tasks {
    let controller = controller.clone();
    let handle = tokio::spawn(async move {
        controller.execute_with_meta(task, executor).await
    });
    handles.push(handle);
}

let results = futures::future::join_all(handles).await;
```

### 3. 异步 LLM 调用

```rust
// 使用异步 LLM 调用避免阻塞
let llm = llm.clone();
tokio::spawn(async move {
    match llm.chat_complete(&messages).await {
        Ok(response) => { /* 处理响应 */ }
        Err(e) => { /* 处理错误 */ }
    }
});
```
